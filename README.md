# Perceptron - a simple example
Show 1 and 2 dimensional examples of perceptron learning algorythm.

Requires python and runs in jupyter

To run on computer
```
python -m venv .venv
.venv/Scripts/activate
pip install -r requirements.txt
jupyter notebook

```

To run on windows
```
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
jupyter notebook

```

A browser should open and show files. Pick one.
